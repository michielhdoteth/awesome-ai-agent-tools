# Multi-Agent Ideation Workflow

**Category:** Evaluation | **Author:** Reddit user (r/ClaudeAI)
**Published:** 2026-06-21 | **Source:** https://www.reddit.com/r/ClaudeAI/comments/1mle79b/

## Use When

You need multiple AI perspectives to produce deeper, more varied design ideas.

## Prompt

> 1. 5 agents with different perspectives and reasoning styles go to work
> 2. Each proposes two designs
> 3. Agents collaborate, share ideas and feedback
> 4. Each goes back to improve design based on shared ideas
> 5. Final design incorporating all feedback

## Verification

- **Title:** Five agents produce diverse designs that are refined through collaboration.
- **Detail:** Final design incorporates feedback from all perspectives.

## Steps

1. Deploy 5 agents with different perspectives and reasoning styles.
2. Each agent proposes two distinct designs.
3. Agents collaborate, sharing ideas and providing feedback.
4. Each agent returns to improve their design based on shared ideas.
5. Produce a final design incorporating all feedback.

## Why

Multiple perspectives produce deeper, more varied ideas than single-agent approach.
