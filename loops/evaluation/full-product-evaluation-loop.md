# The Full Product Evaluation Loop

**Category:** Evaluation | **Author:** Matthew Berman
**Published:** 2026-06-21 | **Source:** https://signals.forwardfuture.ai/loop-library/loops/full-product-evaluation-loop/

## Use When

You need to evaluate every major capability against a consistent quality bar.

## Prompt

> Create [N] realistic scenarios covering every major capability. Before testing, define clear success criteria and choose a consistent evaluation method, such as pass/fail checks or a scoring rubric. Run every scenario under the same conditions and record evidence for each outcome. Fix the underlying cause of anything that does not meet the criteria, rerun the affected scenarios, and then rerun the complete set. Continue until every scenario meets the original quality bar.

## Verification

- **Title:** Every one of the [N] scenarios meets the defined quality bar.
- **Detail:** Evidence is recorded for each outcome.

## Steps

1. List every major capability, define success criteria and evaluation method, choose [N].
2. Run the full set under consistent conditions and evaluate every outcome with evidence.
3. Document each scenario that misses the criteria, fix the underlying issue.
4. Rerun affected scenarios and then the complete set until every outcome meets the bar.

## Why

Fixed capability map makes quality visible across the whole system.
