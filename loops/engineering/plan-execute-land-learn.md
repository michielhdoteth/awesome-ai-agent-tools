**Category:** Engineering | **Author:** iKon85
**Published:** 2026-06-21 | **Source:** https://github.com/iKon85/agent-workflow-kit

## Use When

You need a complete shipping loop for coding agents with planning, execution, landing, and learning phases.

## Prompt

> idea / plan / backlog / raw-issue
>     |
>     v
>   [grill] -> PRD -> to-issues -> atomic issue / wave anchor
>     |
>     v
>   gate -> Execute -> Land -> Learn -> (next idea)

## Phases

- **Plan:** grill-me, grill-with-docs, to-prd, to-issues, board-to-waves
- **Execute:** build, implement
- **Land:** wrapup
- **Learn:** retro, write-a-skill

## Skills

24 total (Plan: 8, Execute: 4, Land: 1, Learn: 2, Setup: 3, Codex review: 4)

## Verification

- **Title:** An idea is grilled into a PRD, broken into issues, implemented, landed, and learned from.
- **Detail:** Each phase has defined skills and completion criteria.

## Steps

1. **Plan:** Grill the idea into a PRD and break it into atomic issues.
2. **Execute:** Implement the approved issues with bounded changes.
3. **Land:** Wrap up and prepare for release.
4. **Learn:** Run retro and write reusable skills.

## Why

Structured phases prevent skipping planning or learning. Skill creation builds institutional knowledge.
