# Prompt Collections

102 curated prompts for AI coding agents across 16 categories. Copy-paste ready for Claude, GPT-4, Gemini, and other models.

## Quick Start

### For Claude Code
```
Copy any prompt below into your Claude Code session.
Example: "Use the Senior Code Reviewer prompt to review my last commit."
```

### For ChatGPT / Gemini
```
Copy the prompt text and paste it into your AI chat.
Example: Paste the "Bug Hunter" prompt, then describe your bug.
```

### For Agent Workflows
```
Chain prompts together for complex tasks.
Example: Use "Codebase Analyzer" first, then "Refactoring Expert" on the results.
```
